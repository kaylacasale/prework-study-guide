Prework Study Guide

HTML

  • Core language of web that deals with content, structure, and meaning of elements.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • Language used to describe the presentation of the website that is created with HTML and how each element is rendered
  • The style element defines the CSS style associated with a website.

Git

  • Git is an open source version control system that allows web developers to collaborate on projects and revert to prior ersions.
  • Use Git version control to create a code repository and save work.
  • Git uses a username to associate any commits (changes) to code with an identity (different username than GitHub).
  • Git commands always start with // git in order to 1) Initiate (or create) a repo 2) Add code to the repo 3) View the versions of code.
  • Add .gitignore file to exlucde files from being tracked.
  • The command //git clone clones the repository down to your local machine.
  • // git status: checks what branch we are currently on.
  • // git checkout -b branch-name: creates a feature branch named "branch-name, "-b" lets you know it is a new branch, "checkout" switches to it.
  • // git add -A command: includes all changes.
  • // git commit -m 'fixed bug in html file header': command pushes commits (changes).
  • // git pull origin main: command will fetch and update only a specific branch called master and origin in the remote repository.
  • // git push origin main branch-name: command is used to push commits to the remote repository.

JavaScript

  • JavaScript is a lightweight programming language that provides basic logic and interactivity for websites.
  • JavaScript functions include variables, control flow, conditional logic, arrays, and [for] loops.
